블로그에 오신 것을 환영합니다!

이 블로그는 **Next.js**와 **Velite**를 사용하여 만들어졌습니다.
MDX를 활용해 포스트를 작성하고, 코드 하이라이팅도 지원합니다.

## 코드 예시

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

## 목록 예시

- Next.js App Router
- Tailwind CSS v4
- Velite (MDX 처리)
- rehype-pretty-code (코드 하이라이팅)
- next-themes (다크모드)

앞으로 다양한 주제로 글을 작성할 예정입니다.
